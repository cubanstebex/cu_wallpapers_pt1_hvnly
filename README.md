## COMMON ISSUES

### Error: You lack the required entitlement to use...
Our files are escrow locked, which means your FiveM server must be connected to the correct Cfx Keymaster account.  
If files are sent through DM, email, or any unauthorized method, the asset may fail to load or your server may crash.

### Error: Model ######## does not exist (not in cd image)
This means your server cannot find the prop in your resource folder. Please make sure:
- The asset was installed correctly
- The resource is started in your TX console
- Your game build version is up to date
- You followed the documentation exactly

### Everything is installed and started, but the furniture does not appear in my menu
This may mean:
- Your `furniture.lua` or `.sql` file was not saved correctly
- The code was placed incorrectly
- The wrong code was entered for your set or script
- Your housing script is not supported
- The documentation was not followed correctly
- Your build or script version is outdated
